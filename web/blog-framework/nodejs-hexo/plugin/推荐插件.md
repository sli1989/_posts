# SEO插件


## 永久链接(permalink)插件

 - hexo-abbrlink

URL的设计尽量做到以下几点：

- 不变性/鲁棒性：`date>title>path>content`，即内容更新最频繁，标题和日期变动较少。
  - ~~path~~，~~content~~ 这俩被pass了
- 唯一性：`content path title date` 唯一性都挺好
- 长度要短 & 容量要大：
  - ~~content~~



Hexo默认采用 date+path作为URL，唯一性很好，但是鲁棒性就比较差（对于我）。像我就经常改变文件的路径，对文件重新归纳整理，这样会造成批量URL变动，严重影响SEO。
