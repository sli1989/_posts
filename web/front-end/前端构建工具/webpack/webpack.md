# 简介

Webpack这个工具可以将你的所有代码和可选择地将依赖捆绑成一个单独
的 .js 文件。


# 安装

```sh
# 全局安装webpack
$ npm install -g webpack

# webpack 已经将 webpack 命令行相关的内容都迁移到 webpack-cli，所以除了 webpack 外，我们还需要安装 webpack-cli：
$ npm install webpack-cli -D -g

# 查看版本
$ npx webpack --version
```

# 配置文件

根目录下新建`webpack.config.js`

webpack


# 打包

```sh
$ sh
```
