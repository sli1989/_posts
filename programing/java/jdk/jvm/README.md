## 自己写jvm

- 如果只是要去“正确地”实现一台 Java 虚拟机，其实并不如大多数人所想的那样高深和困难。
只需要正确读取 Class 文件之中每一条字节码指令，并且能正确执行这些指令所蕴含的操作即可。
